# Como ejecutar el proyecto con Docker
1. Copiar el .env.example a .env
2. Ejecutar el comando docker-compose up --build
3. Acceder a la URL http://localhost:5173
4. Para detener el proyecto, ejecutar el comando: docker-compose down
5. Para borrar los datos, ejecutar el comando: docker-compose down -v
